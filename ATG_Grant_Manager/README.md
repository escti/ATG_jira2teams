# ATG Grant Manager

## Comprehensive Documentation

### Security Best Practices
- Regularly update all dependencies to their latest versions.
- Use environment variables to store sensitive information such as database credentials.
- Implement role-based access control to limit user permissions.

### Installation Guide
1. Clone the repository:
   ```bash
   git clone https://github.com/escti/ATG_Grant_Manager.git
   ```
2. Navigate to the directory:
   ```bash
   cd ATG_Grant_Manager
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Run the application:
   ```bash
   npm start
   ```
